Установка:
- скачайте файл BudjetApp и откройте папку с файлом через IDE систему;
- запустите файл.

Для использования функционала, предлагаемого программой, вводите в консоль номер пункта, указанный перед наименованием функции (по аналогии с пунктом Первый запуск), или
следуйте указаниям в консоли. Числа можно вводить в формате double, также поддерживается проверка на отрицательные числа.

Первый запуск:
- введите в консоль "2" для выбора пункта «2. Зарегистрироваться», создайте аккаунт.
- введите в консоль "1" для выбора пункта «1. Войти», войдите в систему, используя заданный ранее при регстрации логин и пароль.

Работа с финансами:
- добавляйте доходы и расходы.
- устанавливайте бюджеты по категориям.
- просматривайте сводку финансов.
- выполняйте переводы между пользователями.

Завершение работы: выйдите из аккаунта или завершите приложение - данные сохранятся автоматически.
